Detailed Description

Architecture definition of Manchester encoder.

Architecture is implemented using FSM with five states - idle, s0a, s0b, s1a, s1b.

Manchester encoding of input signal d_i is done when v_i signal is set (valid), otherwise output stays equal to zero.
